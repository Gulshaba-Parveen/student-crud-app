# Student CRUD App

A simple **full-stack app** built with **Node.js (Express)** and **React.js** to **create, read, update, and delete student records**. Styled with **Bootstrap** for a responsive UI.  

---

## **Features**

- **Frontend (React.js)**
  - Login page
  - Landing page
  - Create Student form with validation
  - Student List with search, delete, view, and pagination
  - View Student details page

- **Backend (Node.js + Express)**
  - RESTful API: `POST`, `GET`, `PUT`, `DELETE` students
  - SQLite database
  - Simple async/await implementation

## **Installation & Run**

**Backend**
cd backend
npm install
npm run server

**Frontend**
cd frontend
npm install
npm start
Frontend: http://localhost:3000/
Backend: http://localhost:4000/
